형 변환(Casting) 이란?
- 값의 타입(DataType)을 다른 타입으로 변환하는 것
- (DataType)변수;
- 자동형변환(묵시적) : 작은 type이 큰 type에 저장되는 경우 (예:byte->int)
- 강제형변환(명시적) : 큰 type이 작은 type에 저장되는 경우 (예:int->byte, 손실발생), 타입을 강제로 맞춰야 한다.
- boolean type은 형변환이 안 된다.
